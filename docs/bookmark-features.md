# Bookmark 类型功能对照表

NotNow 支持 4 种书签类型：`link`（链接）、`snippet`（片段）、`task`（任务）、`api`（接口）。

## 交互功能

| 功能 | `link` (链接) | `snippet` (片段) | `task` (任务) | `api` (接口) |
|---|---|---|---|---|
| **单击默认动作** | 浏览器打开 URL | 复制内容到剪贴板 | 打开编辑面板 | 打开编辑面板 |
| **⌘+单击默认动作** | 打开编辑面板 | 打开编辑面板 | 打开编辑面板 | 复制 cURL |
| **自定义单击/⌘+Click** | ✅ 可在设置中配置 | ✅ 可在设置中配置 | ❌ 固定为 edit | ❌ 固定逻辑 |
| **卡片级自定义脚本** | ✅ `{url}` / `{TEXT}` | ✅ `{TEXT}` | ❌ | ❌ |
| **右键/详情"打开"** | 浏览器打开 URL | 复制内容 | 打开关联 URL | 复制 cURL |
| **指定打开应用** | ✅ | ✅ | ✅ | ✅ |

## 通用功能

| 功能 | `link` (链接) | `snippet` (片段) | `task` (任务) | `api` (接口) |
|---|---|---|---|---|
| **收藏 / 标签 / 备注** | ✅ | ✅ | ✅ | ✅ |
| **分类 (Category)** | ✅ | ✅ | ✅ | ✅ |
| **封面图 (Cover)** | ✅ | ✅ | ✅ | ✅ |

## 特有元数据

| 类型 | 特有字段 |
|---|---|
| `link` | favicon / coverURL |
| `snippet` | 语言 (`snippetLanguage`) / 格式 (`code` / `quote` / `plain`) |
| `task` | 优先级 (无/低/中/高) / 截止日期 (`dueDate`) / 完成状态 (`isCompleted`) / 过期检测 (`isOverdue`) |
| `api` | HTTP 方法 (GET/POST/PUT/DELETE/PATCH) / Headers / Body / BodyType / QueryParams |

## 导入功能

导入的书签统一创建为 `link` 类型。

| 导入源 | 说明 |
|---|---|
| **Chrome 书签** | 读取本地 Chrome `Bookmarks` JSON 文件，支持多 Profile |
| **GitHub Stars** | 通过 GitHub REST API 拉取指定用户的 starred 仓库 |
| **Twitter (X) Likes** | 通过外部脚本抓取推文，遇到已存在的 URL 自动停止 |
| **NotNow 备份 (.zip)** | 从导出的 zip 恢复书签、分类和配置，支持所有类型 |
| **剪贴板** | 新建书签时自动检测剪贴板中的 URL |

## 批量操作

进入批量模式后，可多选书签执行以下操作（适用于所有类型）：

| 操作 | 说明 |
|---|---|
| **全选 / 清空** | 切换选中当前列表所有书签 |
| **批量移动** | 将选中书签移动到指定分类或"未分类" |
| **批量删除** | 删除所有选中书签 |
| **批量重打标 (AI)** | 使用 AI 重新生成标题、描述和标签 |
